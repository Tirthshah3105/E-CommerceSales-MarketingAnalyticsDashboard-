# E-Commerce Sales & Marketing Analytics Dashboard

## Overview
This project analyzes e-commerce sales and customer transaction data to uncover business insights related to revenue trends, product performance, and customer purchasing behavior. Using **Python, SQL, and Power BI**, the project transforms raw sales data into an interactive analytics dashboard for KPI monitoring and decision-making.

The goal of this project is to help businesses understand:
- how sales are performing over time
- which products and categories drive revenue
- customer purchasing behavior and order patterns
- overall business performance through key sales KPIs

## Objectives
- Analyze e-commerce sales data to identify revenue trends and product performance
- Track important business KPIs such as total sales, profit, quantity sold, and order volume
- Understand customer purchasing patterns and category-level performance
- Build an interactive Power BI dashboard for business reporting and decision-making

## Tech Stack
- **Python** – Data cleaning and exploratory data analysis
- **SQL** – Querying, KPI calculations, aggregations, and business analysis
- **Power BI** – Interactive dashboard creation and visualization
- **Excel / CSV** – Dataset storage and preprocessing

## Project Workflow

### 1. Data Cleaning & Preparation
Performed data cleaning and preprocessing using Python to improve data quality:
- handled missing values
- standardized column formats
- converted date fields
- checked duplicate records
- prepared data for analysis and dashboarding

### 2. Exploratory Data Analysis
Used Python and SQL to analyze:
- total revenue and sales trends
- top-performing products and categories
- customer purchasing patterns
- profit distribution across categories/regions
- order volume and business performance over time

### 3. KPI Calculation
Calculated business KPIs such as:
- Total Sales
- Total Profit
- Total Orders
- Quantity Sold
- Average Order Value
- Category-wise Revenue
- Top Products by Sales

### 4. Dashboard Development
Built an interactive **Power BI dashboard** to visualize:
- overall sales performance
- monthly / yearly revenue trends
- top-performing products
- category and sub-category analysis
- customer and regional insights
- KPI cards for quick business monitoring
